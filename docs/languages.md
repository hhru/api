# Справочник языков


## Получение доступных языков

`GET /languages` возвращает список всех языков.

```json
[
    {
        "id": "ita",
        "name": "Итальянский"
    },
    {
        "id": "nld",
        "name": "Голландский"
    },
    {
        "id": "ell",
        "name": "Греческий"
    }
]
```

### Ошибки

* `403 Forbidden` – ошибка авторизации пользователя.
