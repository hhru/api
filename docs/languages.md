# Справочник языков


## Получение доступных языков

`GET /languages` возвращает список всех языков.

```json
[
    {
        "id": "ita",
        "name": "Итальянский"
    },
    {
        "id": "nld",
        "name": "Голландский"
    },
    {
        "id": "ell",
        "name": "Греческий"
    }
]
```

Пример: [https://api.hh.ru/languages](https://api.hh.ru/languages)
